# Product Launch KPI Dashboard & Risk Management System

## Overview
This project implements a KPI Dashboard and Risk Management System for product launches, designed to help teams track launch progress, monitor risks, and ensure retail readiness. It's built using Python and Streamlit for the dashboard interface, with data analysis capabilities using pandas and numpy.

## Features
- **KPI Dashboard**: Track and visualize key product launch metrics
- **Risk Management**: Monitor and assess potential risks using a risk matrix
- **Retail Readiness Tracking**: Track preparation tasks and inventory levels
- **Cross-Functional Communication**: System for team updates and progress tracking

## Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/product-launch-dashboard.git
cd product-launch-dashboard
```

2. Create a virtual environment (recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install required packages:
```bash
pip install -r requirements.txt
```

## Usage
1. Run the dashboard:
```bash
streamlit run app.py
```

2. Open your browser and navigate to `http://localhost:8501`

## Project Structure
- `data/`: Contains sample datasets and data storage
- `src/`: Source code for the dashboard and risk management system
- `notebooks/`: Jupyter notebooks for development and analysis
- `tests/`: Unit tests
- `app.py`: Main application file

## Technologies Used
- Python 3.8+
- Streamlit
